\chapter{Introduction}
Bienvenue dans le monde de \textbf{Rusttale}. Ce document sert de livre officiel de l'histoire et de guide de l'univers que nous construisons.

\section*{À propos du projet}
Rusttale est un jeu 2D open-source développé avec le langage de programmation Rust et le framework Tetra. C'est un projet communautaire visant à créer une expérience de jeu unique.

\subsection*{Dépôts}
Le projet est divisé en deux dépôts principaux pour séparer le code des ressources :

    \item \textbf{Ressources \& Actifs :} \\
    \url{https://github.com/lazypwny751/resourcetale}
\end{itemize}
